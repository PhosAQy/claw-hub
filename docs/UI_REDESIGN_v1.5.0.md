# 龙虾营地 v1.5.0 - UI 重构说明

## 🎨 新的UI设计

### 首页
- **紧凑的 Agent 列表**：卡片式布局，每个 Agent 显示核心指标
- **核心指标**：
  - 🦞 Agent 名称 + 版本
  - 📊 会话数量
  - 📈 今日 Token
  - 💻 CPU 使用率
  - 🧠 内存使用率
  - 🟢 在线状态

### Agent 详情页（点击展开）
点击任意 Agent 卡片，展开显示专属数据：

1. **📊 Token 消耗柱状图**
   - 最近 6 小时的 Token 消耗
   - 按半小时粒度统计
   - 交互式图表（悬停查看详细数值）

2. **🧩 已加载插件列表**
   - 显示该 Agent 已加载的所有插件
   - 插件名称 + 版本号
   - 卡片式布局，清晰美观

3. **📈 其他统计信息**
   - Gateway 状态
   - 主机信息
   - Agent 版本（支持更新提示）

## ✨ 交互特性

### 展开/收起
- 点击 Agent 卡片头部 → 展开详情
- 再次点击 → 收起详情
- 平滑的动画效果

### 更新提示
- Hub 有更新时，版本号显示为橙色
- Agent 有更新时，显示 "🔄 更新" 按钮
- 点击更新按钮 → 一键远程更新

### 响应式设计
- 桌面端：详情显示为两列（图表 + 插件）
- 移动端：详情显示为单列

## 🚀 更新方法

```bash
# 一键更新
~/claw-hub-update.sh

# 或只更新 Hub
~/claw-hub-update.sh hub
```

## 📝 技术实现

- **前端**: 原生 JavaScript + Chart.js
- **WebSocket**: 实时推送 Agent 状态
- **REST API**: 拉取历史数据（Token 消耗）
- **响应式设计**: CSS Grid + Flexbox
